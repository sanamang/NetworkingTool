# NetworkingTool

A full-stack AI-powered networking platform that automates cold outreach, relationship management, and meeting scheduling using personalized AI agents.Features

AI-Drafted Emails
    - Prompt the Claude API with context and get a personalized cold email drafted instantly, ready to review and send or schedule
Automated Follow-Ups 
    - Configurable per-contact follow-up rules that trigger automatically via a Node.js backend
Response Classification 
    — AI agent classifies replies and escalates ambiguous responses back to the user for review
Calendar & Inbox Integration 
    — Connects to Gmail and Google Calendar to send emails, surface available timeslots, and auto-book meetings
Relationship Management 
    — Logs every interaction, stores call transcripts, and reminds users to reconnect on a custom cadence per contact
Contact Directory 
    — Organizes contacts by company with automatic email prefix/suffix inference
